# Project Title

Simple Express Dotenv

## Description

A simple example on how to use .env file in an Express.js API using dotenv.

## Getting Started

### Dependencies

* Node JS & npm

### Installing

`yarn install` or `npm install`

### Executing program

* `yarn start:dev` or `npm run start:dev`
* go to http://localhost:3333/
